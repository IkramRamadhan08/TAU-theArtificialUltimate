## Contributing

TAU is an open-source project and **contributions are welcome!**

- **Issues & Ideas:** Open a GitHub issue for bugs or feature requests
- **Pull Requests:** Fork the repo, make your changes, and submit a PR
- **Extensions:** Create WASM-based extensions for new languages or tools
- **Documentation:** Help improve guides, docs, and translations

See the [README](../README.md#contributing) for more details.
